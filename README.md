### `>_` about

I build reliability systems and AI agent tooling.

Over the last decade I’ve worked on production infrastructure at scale: SLO platforms for 1,000+ microservices, topology-aware anomaly detection, automated RCA, and JVM/platform migrations across large fleets. Lately I’ve been focused on agent orchestration, developer tooling, and small systems that are useful enough to ship.

### `>_` featured projects

**[`skillpm`](https://github.com/eddieran/skillpm)**
Universal package manager for AI agent skills. Install, sync, and secure skills across Claude Code, Codex, Gemini CLI, Copilot, Cursor, and other runtimes.

**[`lindy-orchestrator`](https://github.com/eddieran/lindy-orchestrator)**
Agent orchestration framework for DAG scheduling, QA gates, worktree isolation, and git-native coordination.

**[`claude-insight`](https://github.com/eddieran/claude-insight)**
Rust-based tooling for inspecting Claude Code sessions, traces, and replay evidence.

**[`moneywiz-ledger`](https://github.com/eddieran/moneywiz-ledger)**
A practical personal-finance skill: turn natural-language expense entries into MoneyWiz deep links with local config and category inference.
